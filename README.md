CS student at USTP (Cagayan de Oro), graduating 2027. I build full-stack apps — mostly the kind with a real-time component, since that's where things actually get interesting to debug.

## Projects

**KABOOOM!** — Real-time multiplayer word game with countdown-based rounds. Custom lobby system for creating/joining rooms with adjustable settings. Built on Django Channels + WebSockets, React (TS) frontend, PostgreSQL. The hard part wasn't the game logic, it was keeping lobby state in sync across clients without janky reconnects.

**Barangay Complaint System** — Full-stack complaint management app for local government use. Real-time Firestore backend, admin auth, protected dashboard. Shipped with an actual CI/CD pipeline — GitHub Actions running 15 unit tests + automated builds + smoke tests, deployed to Vercel with zero manual steps.

**Steezy Gadget Hub** — E-commerce site with dynamic product filtering, infinite scroll, cart, and a mock checkout flow. Auth supports both email/password and anonymous sessions.

## Also doing

Encoding and cross-referencing survey data for a municipal registry (RBIM) — 500+ booklets so far, near-zero error rate. Not glamorous, but it's taught me more about data integrity than any course did.

## Stack

JS/TS, Python, React, Django, PostgreSQL, Firebase, WebSockets/REST. Comfortable with Git/GitHub Actions for CI/CD — used it on more than one project now, not just following a tutorial.
